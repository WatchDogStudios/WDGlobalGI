# WDGlobalGI

Adds a lighter, realtime Global Illumination System for O3DE, that follows [Gaijin Entertainment's Scalable Real-Time Ray Traced Global Illumination for Large Scenes.](https://enlisted.net/en/news/show/25-gdc-talk-scalable-real-time-ray-traced-global-illumination-for-large-scenes-en/#!/)
